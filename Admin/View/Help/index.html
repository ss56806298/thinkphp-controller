1、登录和退出

1.1 登录

输入用户名、密码和验证码
注意：可以勾选“下次自动登录”，方便下次进入后台

1.2 退出

点击“安全退出”，即可退出后台

2、个人资料的修改

在上图圈出的地方，单击，然后填入需要的值即可完成修改

3、英雄的添加和删除

3.1 英雄的添加

点击“添加”按钮，即可完成添加操作
注意：同一个名字的英雄只能添加一个

3.2 英雄的删除

点击“删除”按钮，即可完成删除操作
注意：如果该英雄在队列当中，则无法删除

4、素材的添加和删除

在上图圈出的地方，双击，然后输入你希望拥有素材的数量，即可完成素材的添加，输入“0”，代表删除该素材

5、武器的添加和删除

5.1 武器的添加

点击“添加”按钮，即可完成添加操作
注意：同一个名字的武器只能添加一个

5.2 武器的删除

点击“删除”按钮，即可完成删除操作
注意：如果该武器在队列当中，则无法删除

6、护甲的添加和删除

6.1 护甲的添加

6.2 护甲的删除

点击“删除”按钮，即可完成删除操作
注意：如果该护甲在队列当中，则无法删除

7、头盔的添加和删除

7.1 头盔的添加

点击“添加”按钮，即可完成添加操作
注意：同一个名字的头盔只能添加一个

7.2 头盔的删除

8、快速通关

选择你需要的关卡，比如说“14-4 寒风的低语”，则该关卡前面的所有关卡都会被选中，该关卡后面的所有关卡都不会被选中，
然后点击“提交”按钮
如果选择了“添加相应的精英副本”复选框，除了该关卡自己对应的精英副本外，其他关卡对应的精英副本也会被添加
